Data Collection

There are THREE major types of sources which brokers collect data from which allow them to create user profiles of both online and offline habits.

  • Public sources: public social media profiles, online forums
  • Commercial sources: retailers, subscription lists, financial services, loyalty programs
  • Government sources: census, public records, and social security information

Additionally, many sources collect two distinct types of data: explicit data and implicit data

Explicit Data

Explicit data is data provided directly from the user. This includes feedback surveys or rating. Additionally, many services ask users to explicitly enter personal information like name, age, race, etc. Explicit data may be less reliable as it is dependant on subjective user input.

Implicit Data

Implicit data is data collected about a user's behavior, usually without the user's awareness. This includes watch time, search history, scroll speed, and clicks. Many modern algorithms prefer to rely on implicit data as it is considered more objective than explicit data.
