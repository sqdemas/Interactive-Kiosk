How Do Algorithms Work?

All personalized recommender engines follow the same basic steps:

  • Gather data about a user (input)
  • Analyze the data for patterns
  • Filter based on analysis
  • Provide recommendations to the user (output)
  • See user response and adjust accordingly (feedback loop)

Filtering methods:

  • Content-based
  • Collaborative
  • Hybrid
