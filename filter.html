How Do Algorithms Work?

All personalized recommender engines follow the same basic steps:

  • Gather data as input
  • Analyze for patterns and apply filtering
  • Provide recommendations as output
  • Adjust according to user response — feedback loop

Filtering is where the bulk of the work is done. There are three main methods:

  • Content-based: filtering based on item features such as product description or keywords; helpful when little user data is available
  • Collaborative: compares users against each other to find similarities and recommend similar products
  • Hybrid: combines both content-based and collaborative filtering
