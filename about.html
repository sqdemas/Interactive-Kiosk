About this Project

This project was created for Pitt's ENGCMP 1130 by Samantha Demas. My personal relationship with technology inspired me to pursue this topic in hopes of answering some of the questions that I wondered in my every day life. This project aims to educate about the role of personal data usage in algorithms and the associated privacy risks to fully understand the current state of this paradox.
